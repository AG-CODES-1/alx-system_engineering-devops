# Shell Basics Scripts
This directory contains shell scripts for basic system operations.
### Scripts and Descriptions
# 0. Where am I?
# Description: Prints the absolute path name of the current working directory.

# 1. What’s in there?
# Description: Display the contents list of the current directory.

# 2. There is no place like home
# Description: Changes the working directory to the user’s home directory.

# 3. The long format
# Description: Display current directory contents in a long format.

# 4. Hidden files
# Description: Display current directory contents, including hidden files, in long format.

# 5. I love numbers
# Description: Display current directory contents in long format with user and group IDs displayed numerically and hidden files.

# 6. Welcome
# Description: Creates a directory named my_first_directory in the /tmp/ directory.

# 7. Betty in my first directory
# Description: Moves the file betty from /tmp/ to /tmp/my_first_directory.

# 8. Bye bye Betty
# Description: Deletes the file betty from /tmp/my_first_directory.

# 9. Bye bye My first directory
# Description: Deletes the directory my_first_directory that is in the /tmp directory.

# 10. Back to the future
# Description: Changes the working directory to the previous one.

# 11. Lists
# Description: Lists all files in the current directory, the parent of the working directory, and the /boot directory in long format.

# 12. File type
# Description: Prints the type of the file named iamafile. The file iamafile will be in the /tmp directory when we run the script.

# 13. We are symbols, and inhabit symbols
# Description: Creates a symbolic link to /bin/ls, named __ls__. The symbolic link should be created in the current working directory.

# 14. Copy HTML files
# Description: Copies all the HTML files from the current working directory to the parent of the working directory, but only copies files that did not exist in the parent directory or were newer.

# 15. Let’s move
# Description: Moves all files beginning with an uppercase letter to the directory /tmp/u.

# 16. Clean Emacs
# Description: Deletes all files in the current working directory that end with the character ~.

# 17. Tree
# Description: Creates the directories welcome/, welcome/to/, and welcome/to/school in the current directory.

# 18. Life is a series of commas, not periods
# Description: Lists all the files and directories of the current directory, separated by commas.

# 19. File type: School
# Description: Create a magic file school.mgc that can be used with the command file to detect School data files.
# School data files always contain the string SCHOOL at offset 0.
# Content: 0       string      SCHOOL
